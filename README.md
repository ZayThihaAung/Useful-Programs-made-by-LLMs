# Useful Programs Made by LLMs

Welcome to the **Useful Programs Made by LLMs** repository! 🚀

Here, you'll find a curated collection of useful programs developed with the assistance of advanced language models like Chat GPT, Claude, and Google Gemini. The goal of this project is to showcase the potential of AI in solving complex problems, creating inspiring tools, and enhancing learning.

---

## Technical English Learner App
**Created by:** Zay Thiha Aung, using **Claude**

### Features:
- **Interactive Vocabulary Practice:** Learn technical English quickly with real-world examples.
- **Adaptive Learning:** Customizes lessons based on your progress and interests.
- **Grammar Suggestions:** Provides real-time grammar tips and learning insights.
- **Built-in Quizzes:** Test your knowledge as you learn.

### Example:
The app teaches you the meaning of technical terms used in fields like engineering, programming, and other scientific disciplines. For instance:
- Input the word "debugging" and get: "Finding and resolving issues in a computer program to make it work as expected."

### Credit:
Special thanks to **Claude**, the AI language model, for assisting in building this app.

## Special Relativity Calculator and Lorentz Factor Calculator
**Developed by:** The **MM Pop Science** YouTube channel using **Google Gemini**

### Features:
1. **Special Relativity Calculator:**
   - Calculate time dilation and length contraction under Einstein's Theory of Relativity.
   - Input velocity as a percentage of the speed of light to visualize relativistic effects.

2. **Lorentz Factor Calculator:**
   - Quick computation of the Lorentz factor (γ) based on velocity.
   - Useful for solving relativity-based equations.

### Example:
When you input:
- **Velocity:** 0.8c (80% the speed of light)
You get:
- **Time dilation:** 1.6667 times slower
- **Lorentz Factor (γ):** 1.6667
